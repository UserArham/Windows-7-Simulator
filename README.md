# 🪟 Windows 7 Simulator

Experience the nostalgia of Windows 7 right in your web browser! This simulator replicates the classic desktop environment with interactive apps and a functional Start menu.  

---

## Features
- Windows 7-style desktop and taskbar
- Start menu with functional icons
- Draggable and resizable windows
- Sample apps: Notepad, Calculator, File Explorer
- Nostalgic Windows 7 visual effects

## Usage
- Click the **Start Menu** to access apps
- Drag and resize windows on the desktop
- Minimize, maximize, and close apps like in Windows 7
- Explore included sample applications
